# TaskAssistant
在团队日常工作中，任务和文档分散在多个工具和文件夹中，人工整理耗时长且容易遗漏，导致任务分配不均、协作效率低下以及重复性操作浪费大量时间。尤其对于小型团队来说，每个人都需要手动检查任务状态和文档更新，这增加了管理成本并容易产生错误。

TaskAssistant 由两个内部 Agent 组成，实现多 Agent 协作：分类 Agent 扫描团队任务列表和文档，将新任务自动归类到对应项目模块，并生成初步标签；汇总 Agent 根据分类结果生成每日任务汇总、简要摘要和提醒通知，同时记录每个任务的处理状态。整个流程是多步自动执行的长链逻辑：任务接收 → 自动分类 → 摘要生成 → 提醒发送 → 状态更新，确保每个环节都有反馈和可追踪记录。系统支持通过 Web 界面实时查看任务处理状态、分类结果和提醒记录，使团队成员无需手动整理也能快速掌握整体工作情况。

TaskAssistant 在 5 人团队内落地使用，每天自动处理约 50~100 个新任务或文档，平均节省人工操作时间约 2 小时，任务遗漏率降低约 70%，团队整体协作效率明显提升。系统结构清晰、可扩展，未来计划增加任务优先级排序、自动提醒优化和跨项目协作功能，以进一步提升团队整体效率。
